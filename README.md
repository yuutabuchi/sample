## slimの使い方
- 本家: http://slim-lang.com/

    $ bundle install

- コンパイル方法

    $ bundle exec slimrb html/test.slim > public/test.html

## sassの使い方
- 本家: http://sass-lang.com/guide
- Github: https://github.com/sass/sass
- インストール方法

    $ bundle install

- コンパイル方法

    $ bundle exec sass styles/test.sass > public/test.css
